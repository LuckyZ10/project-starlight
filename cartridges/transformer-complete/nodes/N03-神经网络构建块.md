# 神经网络构建块

> 本节你将学到：神经网络的基本构建块，包括线性变换、梯度下降、神经元和GELU激活函数。

## 核心概念
线性变换 y = Wx + b 是神经网络的基本构建块。它将输入数据通过权重矩阵 W 和偏置 b 进行变换，产生输出数据 y。

## 梯度下降
梯度下降（Gradient Descent）是神经网络训练的核心算法。它通过不断调整网络中的权重和偏置，以最小化损失函数，从而优化模型的性能。

## 神经元
单个神经元是神经网络的原子单元。它接收输入，通过激活函数处理后输出结果。

## GELU激活函数
GELU（Gaussian Error Linear Unit）已成为现代Transformer的标准选择。它是一种平滑的、单调的激活函数，有助于模型性能的提高。

## 代码示例
```python
# 线性变换示例
def linear_transform(x, W, b):
    return W.dot(x) + b

# 梯度下降示例（简化版）
def gradient_descent(W, b, x, y):
    y_pred = linear_transform(x, W, b)
    error = y_pred - y
    W -= error * x
    b -= error

# 神经元示例（使用GELU激活函数）
def gelu(x):
    return 0.5 * x * (1 + torch.tanh(0.01 * (x + 0.01 * x)))

# 假设输入 x, 输出 y, 权重 W 和偏置 b 已定义
# 神经元的输入是线性变换的输出
neuron_input = linear_transform(x, W, b)
neuron_output = gelu(neuron_input)
```

## 常见误区
- 将梯度下降误解为简单的下降过程，而忽略了学习率等参数的重要性。
- 错误地认为所有的激活函数都可以用来构建神经网络。
