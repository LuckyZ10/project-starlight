# 深度学习框架基础

> 本节你将学到：Tensor的关键属性及其在GPU加速中的作用，以及如何通过继承nn.Module基类规范地定义神经网络的结构与计算逻辑。

## 核心概念：Tensor（张量）

Tensor（张量）是深度学习框架中最基础的数据结构。它的本质是一个支持GPU加速的多维数组。与标准的数值计算库（如NumPy）中的数组不同，Tensor专门为深度学习进行了优化，能够极其高效地在GPU上进行大规模矩阵运算。

### 关键属性

要理解和正确使用Tensor，必须掌握它的三个关键属性：

1. **`dtype`**：数据类型属性。它定义了Tensor中存储的数据类型（例如 `torch.float32`、`torch.int64` 等）。明确数据类型有助于控制模型的内存占用和计算精度。
2. **`device`**：设备属性。它指示当前Tensor被分配在哪种计算设备上（例如 `"cpu"` 或 `"cuda"`）。将Tensor的 `device` 设置为GPU是实现深度学习计算加速的核心前提。
3. **`requires_grad`**：是否要求梯度属性。这是一个布尔值。当将其设置为 `True` 时，框架会开始追踪在该Tensor上的所有操作，以便在后续执行反向传播时自动计算出梯度。这极大简化了神经网络参数更新的过程。

**举例**：
如果你有一个包含图像像素数据的多维数组，你需要将其转化为Tensor，并确保其 `device` 为GPU（加速运算），且其对应的网络权重参数的 `requires_grad=True`（以便能够更新权重进行学习）。

## 核心概念：nn.Module 模型基类

在PyTorch等深度学习框架中，构建任何神经网络都必须继承 `nn.Module` 这个模型基类。它提供了一套清晰、规范的模型定义机制，要求开发者严格遵循特定的结构化范式。

### 使用规范

使用 `nn.Module` 定义神经网络时，必须严格遵循以下两步规范：

1. **在 `__init__` 中声明网络层**：`__init__` 是类的初始化函数。在这里，你需要明确网络具备哪些结构组件，例如全连接层、卷积层等。在这里进行的声明相当于为网络准备“积木”。
2. **在 `forward` 中定义计算逻辑**：`forward` 函数定义了数据在网络中流动的具体过程。你需要在此方法中描述在 `__init__` 中声明的各个网络层是如何连接并处理输入数据（即前向传播的数学计算）的。

## 代码示例

以下是Tensor属性检查与 `nn.Module` 使用规范的完整可运行代码：

```python
import torch
import torch.nn as nn

# ==========================================
# 1. Tensor 的基本概念及关键属性
# ==========================================
# 创建一个 2x3 的多维数组（Tensor），并设定关键属性
x = torch.randn(2, 3, dtype=torch.float32, device="cpu", requires_grad=True)

print("=== Tensor 关键属性 ===")
print(f"dtype (数据类型): {x.dtype}")
print(f"device (所在设备): {x.device}")
print(f"requires_grad (需计算梯度): {x.requires_grad}")

# ==========================================
# 2. nn.Module 模型基类的使用规范
# ==========================================
class SimpleNet(nn.Module):
    def __init__(self):
        super(SimpleNet, self).__init__()
        # 规范 1：在 __init__ 中声明网络层
        self.linear1 = nn.Linear(in_features=3, out_features=2)
        self.linear2 = nn.Linear(in_features=2, out_features=1)

    def forward(self, x):
        # 规范 2：在 forward 中定义计算逻辑（前向传播）
        x = self.linear1(x)
        x = torch.relu(x)
        output = self.linear2(x)
        return output

# 实例化网络并进行一次前向传播
model = SimpleNet()
input_data = torch.randn(1, 3)
output = model(input_data)
print(f"\n网络前向传播输出结果: {output}")
```

## 常见误区

- **将网络层放在 `forward` 中定义**：初学者常把 `nn.Linear()` 直接写在 `forward` 方法里。这会导致每次前向传播时都重新创建网络层，不仅极大地拖慢运行速度，还会导致模型参数无法在迭代间保持和更新。**网络结构声明必须在 `__init__` 中完成**。
- **设备不匹配**：在开启GPU加速时，如果输入数据 Tensor 的 `device` 是 `"cpu"`，而模型（`nn.Module`）的参数在 `"cuda"`（GPU）上，框架会报错。必须确保输入数据与模型参数在同一设备上。

## 要点回顾

- **Tensor 基础**：Tensor是支持GPU加速的多维数组，其核心三大属性为 `dtype`（数据类型）、`device`（计算设备）和 `requires_grad`（是否计算梯度）。
- **nn.Module 规范**：继承该基类构建模型时，必须分拆职责——在 `__init__` 中声明所有网络层，在 `forward` 中编写前向传播的计算逻辑。
