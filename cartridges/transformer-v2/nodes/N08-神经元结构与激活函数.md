# 神经元结构与激活函数

> 本节你将学到：神经元中权重与偏置的作用机制，以及 Sigmoid、Tanh、ReLU、GELU 四种常见激活函数的公式、特性与对比。

## 核心概念

神经元是神经网络的基本计算单元。在神经元中，输入信号会与对应的权重相乘并求和，再加上偏置，最后通过一个非线性函数（即激活函数）输出结果。理解参数的物理意义以及激活函数的特性，是构建有效神经网络的基础。

## 详细说明

### 权重与偏置的作用

在神经元的线性变换过程中，计算公式通常表示为 $z = \sum(w_i \cdot x_i) + b$。

- **权重 $w_i$**：其符号和大小决定了输入信号的影响方向和强度。
  - **正权重**：表示正向促进，会增强输入信号对输出的正向影响。
  - **负权重**：表示反向抑制，会削弱或反转输入信号的影响。
  - **绝对值大小**：表示影响程度，绝对值越大，该输入对最终结果的影响力越强。
- **偏置 $b$**：提供“激活阈值”的调节能力。它相当于给神经元的激活条件设定了一个基准线，即使所有输入都为 0，偏置也能决定神经元是否容易被激活。

### 常用激活函数详细对比

激活函数为神经网络引入了非线性，使其能够学习复杂的模式。以下是四种常用激活函数的详细对比：

| 激活函数 | 公式 | 输出范围 | 特性 | 典型应用 |
| :--- | :--- | :--- | :--- | :--- |
| **Sigmoid** | $\frac{1}{1+e^{-x}}$ | $(0,1)$ | 平滑可导，梯度饱和 | 二分类输出层 |
| **Tanh** | $\frac{e^x-e^{-x}}{e^x+e^{-x}}$ | $(-1,1)$ | 零中心化，梯度饱和 | RNN隐藏层 |
| **ReLU** | $\max(0,x)$ | $[0,+\infty)$ | 计算简单，缓解梯度消失 | 隐藏层默认选择 |
| **GELU** | $x \cdot \Phi(x)$ | $(-\infty,+\infty)$ | 平滑可导，Transformer首选 | Transformer FFN |

*(注：$\Phi(x)$ 表示标准正态分布的累积分布函数)*

### 梯度饱和问题与缓解机制

在训练深层网络时，激活函数的导数（梯度）特性直接影响模型能否有效更新权重：

- **Sigmoid 和 Tanh 的局限**：这两个函数都存在**梯度饱和问题**。当输入值 $x$ 非常大或非常小的时候，函数曲线变得非常平缓，导致其导数（梯度）趋近于 0。在反向传播中，这种极小的梯度不断相乘，很容易导致底层网络权重无法更新，即“梯度消失”现象。
- **ReLU 的应对**：ReLU 通过极其简单的计算（正区间导数为 1）极大地缓解了梯度消失问题，使得深层网络的训练成为可能。
- **GELU 的优化**：作为较新的激活函数，GELU 具备**平滑可导**的特性，在缓解梯度消失问题的同时，避免了 ReLU 硬转折点带来的负半轴梯度完全死掉的问题，因此在 Transformer 架构中表现优异。

## 代码示例

以下是使用 Python 和 NumPy 实现并计算上述四种激活函数的代码：

```python
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def tanh(x):
    return (np.exp(x) - np.exp(-x)) / (np.exp(x) + np.exp(-x))

def relu(x):
    return np.maximum(0, x)

# 近似计算 GELU (x * Phi(x))
def gelu(x):
    return 0.5 * x * (1 + np.tanh(np.sqrt(2 / np.pi) * (x + 0.044715 * np.power(x, 3))))

# 测试数据
x = np.array([-10, -1, 0, 1, 10])

print("Sigmoid:", sigmoid(x))
print("Tanh:   ", tanh(x))
print("ReLU:   ", relu(x))
print("GELU:   ", gelu(x))
```

## 常见误区

- **误区 1：在隐藏层默认使用 Sigmoid**。由于 Sigmoid 存在严重的梯度饱和问题且输出不是零中心化的，极易导致梯度消失，现在极少用于神经网络的隐藏层，通常仅用于二分类的最终输出层。
- **误区 2：忽略偏置的作用**。虽然权重主导了特征提取，但如果没有偏置 $b$，分类超平面将被强制限制在必须经过原点，这会极大降低模型的拟合能力。

## 要点回顾

- 权重 $w_i$ 的符号决定影响方向（正促进、负抑制），绝对值决定影响程度；偏置 $b$ 负责调节激活阈值。
- Sigmoid 输出范围 $(0,1)$ 适用于二分类；Tanh 输出 $(-1,1)$ 常用于 RNN；ReLU 计算简单，是隐藏层的默认选择；GELU 平滑可导，是 Transformer 的首选。
- Sigmoid 和 Tanh 在输入值极大或极小时会遇到梯度饱和问题；ReLU 通过计算简单缓解了梯度消失，而 GELU 则通过平滑可导的特性进一步优化了这一过程。
