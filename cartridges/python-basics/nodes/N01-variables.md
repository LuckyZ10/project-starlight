# 变量与赋值

## 核心概念

变量是存储数据的"盒子"。在 Python 中，你不需要提前声明变量类型——直接赋值即可。

```python
name = "Alice"
age = 25
height = 1.68
```

上面三行分别创建了三个变量：`name` 存了一个字符串，`age` 存了一个整数，`height` 存了一个小数。

## 赋值的本质

Python 中的 `=` 不是数学中的"等于"，而是**赋值**：把右边的值放进左边的变量里。

```python
x = 10       # x 现在是 10
x = 20       # x 被重新赋值为 20，旧的 10 被丢弃
y = x        # y 现在也是 20
```

## 多重赋值

Python 支持一次性给多个变量赋值：

```python
a, b, c = 1, 2, 3
x = y = z = 0
```

## 常见误区

1. **变量未定义就使用**：`print(score)` 会报 NameError，必须先赋值。
2. **混淆 `=` 和 `==`**：`=` 是赋值，`==` 是比较。
3. **变量名不合法**：`2nd_place = "Bob"` 会报语法错误，变量名不能以数字开头。

## 命名规范

- 使用小写字母和下划线：`user_name`、`total_score`
- 见名知意：避免用 `a`、`b`、`c` 这样的名字（除非是临时变量）
- 避免使用 Python 关键字：`if`、`for`、`class` 等不能做变量名
